## API Contracts

### GET /api/outage-transcripts
```typescript
// Request
?zip=06101&hours=24

// Response
{
  zip_code: "06101",
  time_range_hours: 24,
  call_count: 45,
  transcripts: [
    {
      call_id: 12345,
      customer_id: "CUST-001",
      started_at: "2025-11-18T14:30:00Z",
      ended_at: "2025-11-18T14:38:30Z",
      duration_minutes: 8.5,
      call_reason: "technical_support",
      transcript: "Agent: Hello...",
      customer_location: "Hartford, CT",
      service_address: "123 Main St, Hartford, CT 06101"
    },
    // ... more transcripts
  ]
}
```

### POST /api/outage-chat
```typescript
// Request
{
  zip: "06101",
  hours: 24,
  messages: [
    { role: "user", content: "What are the common issues?" }
  ]
}

// Response
{
  reply: "Based on the 45 call transcripts...",
  contextUsed: {
    transcriptCount: 45,
    zipCode: "06101",
    timeRangeHours: 24
  }
}
```
